# Sample Input

我以前总以为，写公众号最难的是观点不够新。后来做久了才发现，真正难的不是“想不到”，而是“写出来不像自己”。

你会发现一件很别扭的事。脑子里的判断明明挺鲜活，真落到稿子上，却像被热水烫平了一样，句子一排排站得过分整齐。看起来没有错，甚至还挺完整。可就是不对劲。

说实话，这种不对劲，读者是能感觉到的。哪怕他说不出哪里假，也会下意识划走。因为人写东西，多少带点脾气，带点迟疑，带点当时那个瞬间的体温。太圆，反而像塑料。

所以后来我改方法了。我不再一上来就追求完整，而是先把自己当时真正在想什么记下来。哪怕只有两句，哪怕还很毛糙。先把那个“人”留下，再考虑怎么把它整理成能发出去的东西。
