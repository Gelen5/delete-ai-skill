# Selling Guide

## Best product positioning

Sell this as:

- personal style writing skill
- low-AI-tone rewriting toolkit
- author voice cloning workflow for self-media content

Do not sell this as:

- guaranteed detector bypass
- 100 percent pass tool
- anti-detection cheat product

## Stronger product promise

Recommended promise:

- make AI-heavy drafts sound more personal
- reduce templated writing patterns
- turn past articles into reusable style assets
- improve consistency for public-account and self-media writing

## Suggested offer stack

### Base package

- the skill
- style DNA extractor
- QA checklist
- scenario templates

### Premium package

- base package
- onboarding call
- 1 to 3 custom style DNA builds
- rewrite examples

### Team package

- premium package
- multi-author style library
- content workflow SOP
- internal training material

## Suggested buyer types

- self-media creators
- ghostwriters
- content studios
- public-account operators
- founders writing personal-brand content

## Demo angle

The easiest demo is:

1. show an AI-heavy draft
2. show the style sample
3. run the workflow
4. show the rewritten version
5. explain the visible changes in tone, rhythm, detail, and platform fit

## Sales disclaimer

Use language like:

> This product improves style naturalness and personal voice consistency. It does not guarantee any third-party detector result, and outcomes may vary with model choice, prompt quality, and platform updates.
